# KP Tubes & Nxtora Distribution Portal

This project is a **frontend prototype landing page** developed as part of my internship work at **KP Tubes Pvt Ltd**.

The goal of this project is to design a **modern and professional landing page** that represents two business entities:

- **KP Tubes Pvt Ltd** – Industrial pipes, tubes, and steel products supplier.
- **Nxtora** – Authorized Castrol lubricant dealership.

This portal acts as a **unified distribution platform** showcasing both industrial products and automotive lubricants.

---

## Project Objective

The objective of this prototype is to:

- Design a **corporate landing page UI**
- Represent **two related business units on one platform**
- Provide company information and contact details
- Create a clean frontend structure that can later be extended into a full web application

---

## Features

- Professional **hero landing section**
- Business showcase for **KP Tubes Pvt Ltd**
- Business showcase for **Nxtora Castrol Dealership**
- Corporate styled **About section**
- Contact information section
- Responsive layout
- Clean and modern UI
